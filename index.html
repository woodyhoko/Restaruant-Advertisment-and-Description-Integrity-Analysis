NLP · Information Integrity

Does the menu
match the reviews?

Restaurant platforms let operators write glowing self-descriptions next to unfiltered customer reviews — a systematic information asymmetry. This project audits the operator's own claims, quantifying how well a description is corroborated (or contradicted) by the review corpus.

the integrity score
Three NLP signals, one composite.
S=(1−w₁)·cos(D,R)+w₂·(1−|Δsentiment|)+w₃·(1−JSD)
how it's measured
Cosine, sentiment, topics.
🔗

TF-IDF cosine

Vectorize description and pooled reviews; distinctive claims ("michelin", "authentic Sichuan") dominate over filler.

weight 0.4
😐

Sentiment gap

VADER compound sentiment of the ad minus the reviews. A glowing description over tepid reviews = overclaiming.

weight 0.3
🧩

Topic divergence

LDA topic distributions compared by Jensen-Shannon divergence — "romantic fine dining" vs "fast food, long wait".

weight 0.3
🔍

What it finds

High-integrity listings use specific, verifiable claims; chains beat independents; sentiment gap > 0.3 is the strongest red flag.
